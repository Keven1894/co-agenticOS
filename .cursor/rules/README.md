## ðŸ”§ How Rules Work

### Auto-Loading
Cursor automatically loads all rules in this directory when the project is opened. No manual configuration needed.

### Priority System
Rules are applied based on priority:
1. **Critical** - Absolute requirements (language standards)
2. **Highest** - Core identity and behavior
3. **High** - Quality and process standards

### Context Integration
Rules are merged into Cursor's system context and applied intelligently based on:
- File type being edited
- Current task context
- Project structure
- Content being created
